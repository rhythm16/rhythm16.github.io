這篇是靠感覺打的，沒有實作跟去check組合語言，因為我覺得觀念對了，真的有不確定的地方再試就好。

static

static代表符號只在這個編譯單元有效，編譯單元就是一個.c檔在預處理器處理之後的東東(include, define那些被展開之後)。

inline

inline則是建議編譯器可以把函式內嵌在呼叫的地方，節省資源提升速度。
